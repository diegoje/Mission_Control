# Mission Control

This repo documents and automates **Mission Control** for MedForm3D.

**Canonical sources**
- **Notion**: Projects, Contacts, SOPs, Ideas, long‑form context
- **Todoist**: day‑to‑day tasks and reminders
- **Gmail/Calendar**: inputs for planning and scheduling

**Autonomy rules (agreed)**
- Read access: Gmail + Calendar **OK**
- Write access: Notion + Todoist **OK**
- Email sending: **never** unless explicitly requested
- Daily summaries: **only on request**, delivered on Telegram

## Structure
- `docs/mission_control_overview.md`
- `docs/notion_schema.md`
- `docs/todoist_rules.md`
- `docs/email_calendar_workflow.md`
- `docs/automation_runbook.md`

## Security
- No passwords/keys stored in repo
- `.gitignore` excludes `.env`, `.ssh/`, memory logs, etc.
