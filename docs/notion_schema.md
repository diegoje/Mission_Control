# Notion Schema (Mission Control)

Mission Control uses Notion as the canonical source for Projects, Contacts, SOPs, Ideas, and long‑form context.

## Databases
- **Projects DB**
  - URL: https://www.notion.so/30b5d4551f5681869fa7c256294c7e62
  - Properties: Name (title), Status (select), Priority (select), Due (date), Tags (multi‑select), Todoist Project ID (rich_text), Company (rich_text)
  - Company values: MedForm3D, RCJU, Abbott, Roche, Biogen, Other

- **Tasks DB (Todoist mirror)**
  - URL: https://www.notion.so/30b5d4551f5681bb865be9f6dbd823ff
  - Properties: Name (title), Status, Assigned To, Due, Priority, Todoist Task ID, Source

- **Contacts DB**
  - URL: https://www.notion.so/30b5d4551f568130b4dce36cd453d0b2
  - Properties: Name (title), Organization, Role, Email, Phone, Notes

## Rules
- All new entries must be created under Mission Control.
- No PHI/confidential data without explicit approval.
