## Deliverable types

- Short-form video script (Reels/TikTok): hook → problem → proof → CTA
- Carousel outline: slide-by-slide
- LinkedIn post: 1 strong idea + CTA + 3 hashtags
- Image prompt pack: 3–5 prompts + aspect ratio + style notes + negative prompts
